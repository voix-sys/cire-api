# Product Hunt Launch Copy — CIRE

## Tagline
Deterministic INCI safety scoring API for AI agents

## Description (short)
Turn cosmetic ingredient lists into explainable safety JSON. Built for shopping agents, compliance pipelines, and K-beauty recommendation engines.

## Description (long)
CIRE is a deterministic, rule-based API that scores INCI strings into category-level safety risk outputs.
- `/v1/analyze` for core risk scoring
- `/v1/synergy` and `/v1/optimal` (Pro) for formulation intelligence
- Consistent JSON output for production workflows
